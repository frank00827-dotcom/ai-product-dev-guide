# 附录 A1 - Prompt 模板库

> 全阶段 Prompt 模板速查，按使用场景分类

## 使用说明

- `[变量]` 表示需要替换为实际内容的占位符
- 每个模板标注了推荐使用的 AI 工具
- 质量评级基于团队实际使用反馈，定期更新

## 五、代码审查类

### R-01: 通用代码审查

```
工具: Claude Code  |  质量评级: ⭐⭐⭐⭐⭐

# 角色
你是资深代码审查专家。

# 审查维度（按优先级）
1. 安全性 — 注入、XSS、敏感数据泄露、权限绕过
2. 正确性 — 业务逻辑、边界条件
3. 性能 — N+1查询、内存泄漏
4. 可维护性 — 命名、模块边界、硬编码

# 输出格式
每个问题:
- 严重程度: 🔴 必须修复 / 🟡 建议修复 / 🟢 可选优化
- 文件:行号
- 问题描述
- 修复建议（附代码）
```

### R-02: 医疗合规专项审查

```
工具: Claude Code  |  质量评级: ⭐⭐⭐⭐

# 角色
你是医疗信息安全审查专家。

# 审查重点
1. 敏感字段（姓名/手机/身份证/诊断）是否加密存储
2. API 响应是否包含不应暴露的字段
```
